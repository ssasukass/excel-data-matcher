# Excel 数据匹配填充工具

一个灵活的 Python 工具，用于在两个 Excel 表格之间按指定列匹配数据并自动填充，支持模糊匹配、复合键、缺失记录追加等功能。

---

## 功能

- **按任意列匹配** — 指定一个或多个列作为索引，从源表匹配数据并填充到目标表
- **列名映射** — 源表和目标表列名不同时，支持 `源列名:目标列名` 格式映射
- **多列复合索引** — 支持多个列组合匹配（如姓名 + 院校/系），精准区分重名
- **字段级模糊匹配** — 复合键下各字段独立计算相似度取平均，解决跨表数据细微差异
- **缺失记录追加** — 将源表有但目标表没有的记录作为新行追加到输出表底部
- **自动打包依赖** — xlrd 库直接打包在 `_lib` 目录，无需额外安装即可读取 .xls 文件
- **表头行适配** — 文件第一行是标题时，可指定表头所在行号
- **自动备份** — 匹配前自动备份目标文件
- **权限回退** — 输出目录无写入权限时自动保存到系统临时目录
- **图形界面** — 提供 Tkinter 图形界面，免命令行操作
- **GBK 兼容** — 支持含生僻字的姓名，自动处理编码问题

## 快速上手（图形界面）

1. 双击 `match_excel_gui.py` 打开图形界面
2. 在"源表"选择有完整数据的 Excel 文件
3. 在"目标表"选择待填充的 Excel 文件
4. 在"索引列"输入匹配依据的列名（如 `姓名`）
5. 在"填充列"输入要复制的列名（如 `学号`, `班级`）
6. 点击 **▶ 开始匹配**

> 如果文件第一行是标题（如"2001级"），需要把"表头行"设为 `1`。

## 命令行用法

```
python match_excel.py -s 源表.xlsx -t 目标.xlsx -k 索引列 -f 填充列
```

### 基本匹配

```bash
# 按姓名匹配，填入学号和班级
python match_excel.py -s 源表.xlsx -t 目标.xlsx -k 姓名 -f 学号 -f 班级
```

### 多列索引

```bash
# 按姓名+专业匹配，避免重名干扰
python match_excel.py -s 源.xlsx -t 目标.xlsx -k 姓名 -k 专业 -f 学号 -f 班级
```

### 列名映射

```bash
# 源表列名英文，目标表列名中文
python match_excel.py -s src.xlsx -t tgt.xlsx -k Name:姓名 -f StudentID:学号
```

### 模糊匹配

```bash
# 开启模糊匹配，处理姓名/院系的细微差异
python match_excel.py -s 源.xlsx -t 目标.xlsx -k 姓名 -k 学院:院校/系 -f 学号 -f 班级 --fuzzy --threshold 0.85
```

### 表头行

```bash
# 源表第一行是标题，真正表头在第二行
python match_excel.py -s 2001级2.xls -t 2005.xlsx -k 姓名 -f 学号 --source-header-row 1
```

### 缺失记录追加

```bash
# 匹配 + 将源表有但目标表没有的人追加到输出表
python match_excel.py -s 源.xlsx -t 目标.xlsx -k 姓名 -f 学号 --append-missing
```

> 追加时自动将源表中所有能对应的列（列名相同或已映射）一起带过去，已有记录不受影响。

### 完整参数

| 参数 | 说明 |
|---|---|
| `-s, --source` | 源表路径（必填） |
| `-t, --target` | 目标表路径（必填） |
| `-o, --output` | 输出文件路径 |
| `-k, --key` | 索引列，可多次指定 |
| `-f, --fill` | 填充列，可多次指定 |
| `--source-sheet` | 源表 sheet 名称 |
| `--target-sheet` | 目标表 sheet 名称 |
| `--source-header-row` | 源表表头行号（0起始） |
| `--target-header-row` | 目标表表头行号（0起始） |
| `--fuzzy` | 启用模糊匹配 |
| `--threshold` | 模糊匹配阈值（默认 0.85） |
| `--append-missing` | 将源表缺失记录追加到输出 |
| `--no-backup` | 不创建备份文件 |

## 高级功能详解

### 复合键模糊匹配

当使用多个索引列（如 `姓名 + 院校/系`）时，模糊匹配会分别计算每个字段的相似度再取平均值，而非把整个键当作一个字符串去比较。例如：

```
目标表：闫剑 / 光电工程学院
源表：  闫剑 / 光电学院
→ 姓名相似度: 1.0   院系相似度: 0.8
→ 平均得分: 0.9 → 匹配成功（超过阈值 0.85）
```

相比全局字符串比较（得分仅 0.762），字段级平均相似度更准确地反映了实际匹配程度。

### 按姓名预过滤优化

复合键模糊匹配时，先按姓名精确过滤再比对院系字段，运行时间从几十秒优化到 3 秒以内。

### 缺失记录追加

`--append-missing` 会将源表中现有目标表里没有的记录（按复合键判断）追加到输出表底部。追加时会自动扫描源表所有列，将能与目标表对应的列（列名相同或已在键/填充映射中）一并带过去。

## 常见问题

**Q：匹配结果全是空的？**
A：大概率是表头行没设置对。文件第一行是标题时，需要将 `--source-header-row` 设为 `1`。

**Q：列名带冒号没被解析？**
A：确保使用半角冒号 `:`，而不是全角冒号 `：`。脚本已支持两种冒号，但建议使用半角。

**Q：报错 `No module named 'xlrd'`？**
A：工具包 `_lib` 目录已预装 xlrd。如果还是报错，可手动安装：`pip install xlrd`

**Q：控制台输出乱码？**
A：Windows 控制台默认 GBK 编码，脚本已自动切换为 UTF-8 输出。

**Q：输出文件保存失败？**
A：如需写入的目录权限受限，脚本会自动回退到系统临时目录（`%TEMP%`），拿到文件后手动复制到目标位置即可。

**Q：如何把工具发给别人用？**
A：将整个 `Excel匹配工具` 文件夹压缩打包，接收方解压后安装 Python 和依赖即可使用。
