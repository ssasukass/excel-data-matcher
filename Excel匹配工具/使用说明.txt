Excel Data Matcher - 使用说明
====================================

一、这个工具能干什么？
─────────────────────

把两个 Excel 表格按指定列匹配起来，把源表的数据填充到目标表。

举个例子：
  你有两个表：
    表A（源表）：有【姓名、学号、班级、学院】等完整信息
    表B（目标表）：只有【姓名】，学号和班级是空的

  这个工具能按【姓名】把两个表对上号，
  把表A里的【学号】和【班级】填到表B对应的行里。


二、准备工作
───────────

1. 安装 Python（如果没有的话）
   - 下载地址：https://www.python.org/downloads/
   - 下载后安装，安装时勾上 "Add Python to PATH"
   - 装完打开命令提示符（cmd），输入 python --version
     能看到版本号就说明装好了

2. 安装依赖（只需一次）
   打开命令提示符，输入：
   
     pip install pandas openpyxl
    
   等它装完即可。


三、文件清单
───────────

拿到工具包后，解压到任意文件夹，里面应该有：

  match_excel.py          核心脚本（命令行用）
  match_excel_gui.py      图形界面（推荐）
  _lib/                   xlrd 库（用来读 .xls 旧格式文件）
  requirements.txt        依赖清单
  启动工具.bat            一键启动（双击即可）


四、快速上手（图形界面 - 推荐）
─────────────────────────────

双击 "启动工具.bat" 或 match_excel_gui.py，界面如下：

步骤 1：选择文件
  - 【源表】点击浏览 → 选有完整数据的 Excel 文件
  - 【目标表】点击浏览 → 选需要填充的 Excel 文件
  - 【输出】不填的话默认在目标表同目录生成 "已填充_原文件名.xlsx"

步骤 2：设置 Sheet 和表头行
  - Sheet 下拉框会自动识别，选对 sheet 即可
  - 表头行默认为 0（第一行就是列名）
  - 如果第一行是标题（如"2001级"），表头行要改成 1

步骤 3：设置索引列（匹配依据）
  - 默认示例为"姓名"
  - 可点 "+ 添加索引列" 增加多列组合匹配
  - 输入格式：纯列名（如 姓名）
  - 列名不同时：源列名:目标列名（如 Name:姓名）

步骤 4：设置填充列（要复制的内容）
  - 点 "+ 添加填充列" 增加
  - 格式同上
  - 如果填充列在目标表中不存在，会自动新建

步骤 5：点击 "▶ 开始匹配"
  - 日志区会实时显示进度和结果
  - 完成后会在目标表同目录生成输出文件


五、命令行用法（进阶）
────────────────────

基础匹配（按姓名匹配，填入学号和班级）：

  python match_excel.py -s 源表.xlsx -t 目标.xlsx -k 姓名 -f 学号 -f 班级

多列索引 + 多列填充：

  python match_excel.py -s 源.xlsx -t 目标.xlsx -k 姓名 -k 专业 -f 学号 -f 班级 -f 学院

列名不同时（源表英文，目标表中文）：

  python match_excel.py -s src.xlsx -t tgt.xlsx -k Name:姓名 -f StudentID:学号 -f Class:班级

第一行是标题的情况：

  python match_excel.py -s 2001级2.xls -t 2005.xlsx -k 姓名 -f 学号 --source-header-row 1

模糊匹配（处理姓名有小差异的情况）：

  python match_excel.py -s 源.xlsx -t 目标.xlsx -k 姓名 -f 学号 --fuzzy --threshold 0.85

所有命令行参数：

  -s, --source        源表路径（必填）
  -t, --target        目标表路径（必填）
  -o, --output        输出文件路径
  -k, --key           索引列，可多次指定（必填）
  -f, --fill          填充列，可多次指定（必填）
  --source-sheet      源表 sheet 名称（默认自动检测）
  --target-sheet      目标表 sheet 名称（默认自动检测）
  --source-header-row 源表表头行号，0起始（默认0）
  --target-header-row 目标表表头行号，0起始（默认0）
  --fuzzy             启用模糊匹配
  --threshold         模糊匹配阈值（默认0.85）
  --no-backup         不创建备份


六、常见问题
───────────

Q：打开后报错 "No module named 'xlrd'"？
A：这是因为你的文件是 .xls 旧格式。工具包里已经带了 xlrd，
   如果还报这个错，在命令行运行：
     pip install xlrd

Q：匹配不上（显示大量未匹配）？
A：检查以下几点：
   1. 表头行号是否设对？第一行是标题时需改为 1
   2. 索引列的列名是否拼写正确？
   3. 源表和目标表的列名是否一致？不一致时要用 "源:目标" 格式
   4. 姓名前后是否有空格？

Q：匹配结果全是空值？
A：同上，大概率是表头行设置不对，导致列名没读出来。

Q：.xls 文件打不开？
A：这个工具兼容 .xls 旧格式和 .xlsx 新格式。
   如果 .xls 文件打不开，可以用 Excel 打开后另存为 .xlsx 格式。

Q：匹配结果能保留目标表原有的其他列吗？
A：可以。工具只修改你指定的填充列，其他列原样保留。

Q：如何把工具发给别人用？
A：直接把整个文件夹（包括 _lib 子目录）压缩打包发过去。
   对方解压后需要安装 Python，然后双击 "启动工具.bat" 即可。


七、输出文件说明
───────────────

匹配成功后会在指定位置生成输出文件（默认在原目录生成 "已填充_xxx.xlsx"），
同时自动备份原始目标文件（"备份_xxx.xlsx"）。

输出文件 = 目标表原始数据 + 填充列已补全


如果使用过程中遇到其他问题，可以按以下方式排查：
1. 查看运行日志区的红色/错误提示
2. 检查表头行号设置
3. 确认列名拼写是否完全一致（包括空格和标点）
